How to add a new topic

  1. Create topics/YYYY-MM-topic-name/.
  2. Write 01-..., 02-..., ... numbered lessons.
  3. Drop a topic landing index.html in the same folder.
  4. Add a card to the grid above.
  5. Push to main. GitHub Pages redeploys in ~30s.
Tip: bookmark this page. New topics land at the top of the grid as they're added.
